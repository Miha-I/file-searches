Configuration file structure


[config]
show_folder=false
long_names=false
privilege=false
version=1


show_folder - Writing folders in directories (may increase the size of the log file)
long_names - File name support more than 260 characters (up to 32000 supports version 2)
privilege - !!!!! Work with Backup privileges (log file size can be more than 1 Gb)
version - search version
